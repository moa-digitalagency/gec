# Notes d'Installation
# --------------------
# Pour installer toutes les dépendances:
# 1. Créez un environnement virtuel: python -m venv venv
# 2. Activez l'environnement:
#    - Linux/Mac: source venv/bin/activate
#    - Windows: venv\Scripts\activate
# 3. Installez les dépendances: pip install -r project-dependencies.txt
#
# Variables d'environnement requises (fichier .env):
# DATABASE_URL=postgresql://user:password@localhost/gecmines
# SESSION_SECRET=your-secret-key-here
# GEC_MASTER_KEY=your-encryption-key
# GEC_PASSWORD_SALT=your-password-salt
