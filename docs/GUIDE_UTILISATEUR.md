# Guide Utilisateur - GEC (Gestion Électronique du Courrier)

## Introduction

Le GEC est un système de gestion électronique du courrier destiné aux institutions gouvernementales et entreprises. Il permet d'enregistrer, suivre et archiver les courriers entrants et sortants de manière sécurisée.

---

## Connexion au Système

### Accès Initial

1. Ouvrir l'application dans un navigateur web
2. Saisir votre nom d'utilisateur dans le champ "Username"
3. Saisir votre mot de passe dans le champ "Password"
4. Cliquer sur "Login"

### Compte Administrateur Par Défaut

- Nom d'utilisateur : `sa.gec001`
- Mot de passe : `TempPassword123!` (à modifier immédiatement après première connexion)

### Sécurité de Connexion

- Après 8 tentatives échouées, votre adresse IP sera temporairement bloquée pendant 15 minutes
- Les mots de passe doivent contenir au minimum 8 caractères, une majuscule, une minuscule, un chiffre et un caractère spécial

## Enregistrement d'un Courrier

### Courrier Entrant

1. Aller dans "Enregistrer un courrier"
2. Sélectionner le type "Courrier Entrant"
3. Remplir les champs obligatoires :
   - **Objet** : description du courrier
   - **Expéditeur** : nom de l'organisme ou personne expéditrice
   - **Copie au Secrétaire Général** : Oui/Non
   - **Date de rédaction** : date inscrite sur le courrier
   - **Statut** : état du courrier (Reçu, En cours, Traité, etc.)
4. Optionnel : joindre un fichier (PDF, images)
5. Cliquer sur "Enregistrer"

Le système génère automatiquement un numéro d'accusé de réception unique.

### Courrier Sortant

1. Aller dans "Enregistrer un courrier"
2. Sélectionner le type "Courrier Sortant"
3. Remplir les champs obligatoires :
   - **Objet** : description du courrier
   - **Destinataire** : nom de l'organisme ou personne destinataire
   - **Type de courrier sortant** : Note circulaire, Lettre officielle, Mémorandum, etc.
   - **Date d'émission** : date de création du courrier
   - **Numéro de référence** : référence interne (optionnel)
4. Ajouter des informations complémentaires si nécessaire
5. Joindre le document (optionnel)
6. Cliquer sur "Enregistrer"

## Recherche de Courriers

### Recherche Simple

1. Utiliser la barre de recherche en haut de page
2. Saisir un terme (numéro, objet, expéditeur, destinataire)
3. Les résultats s'affichent automatiquement

### Recherche Avancée

1. Aller dans "Recherche"
2. Filtrer par :
   - Type de courrier (Entrant/Sortant)
   - Période (date de début, date de fin)
   - Statut
   - Expéditeur/Destinataire
3. Cliquer sur "Rechercher"

---

## Gestion des Statuts

Les statuts disponibles par défaut :

| Statut | Description |
|--------|-------------|
| RECU | Courrier reçu, non encore traité |
| EN_COURS | Courrier en cours de traitement |
| TRAITE | Courrier traité |
| ARCHIVE | Courrier archivé |
| URGENT | Courrier nécessitant une attention immédiate |

### Modification du Statut

1. Ouvrir le détail du courrier
2. Cliquer sur "Modifier"
3. Sélectionner le nouveau statut
4. Enregistrer

---

## Transmission de Courriers

La fonction de transmission permet de transférer un courrier vers un autre utilisateur ou département.

### Transmettre un Courrier

1. Ouvrir le détail du courrier
2. Cliquer sur "Transmettre"
3. Sélectionner le destinataire (utilisateur ou département)
4. Ajouter un commentaire (optionnel)
5. Joindre un document supplémentaire (optionnel)
6. Cliquer sur "Envoyer"

### Suivi des Transmissions

- L'historique des transmissions est visible dans le détail du courrier
- Le destinataire reçoit une notification
- L'état de lecture est affiché (lu/non lu)

## Notifications

### Types de Notifications

- Nouveau courrier enregistré
- Courrier transmis à vous
- Modification de statut
- Alertes système

### Consultation

1. Cliquer sur l'icône de notification (cloche)
2. Visualiser la liste des notifications
3. Cliquer sur une notification pour voir le détail

---

## Gestion du Profil

### Modifier son Profil

1. Cliquer sur votre nom en haut à droite
2. Sélectionner "Mon Profil"
3. Modifier les informations souhaitées :
   - Photo de profil
   - Nom complet
   - Email
   - Langue préférée (Français/Anglais)
4. Enregistrer

### Changer son Mot de Passe

1. Aller dans "Mon Profil"
2. Saisir l'ancien mot de passe
3. Saisir le nouveau mot de passe (respecter les critères de sécurité)
4. Confirmer le nouveau mot de passe
5. Enregistrer

---

## Corbeille

Les courriers supprimés sont placés dans la corbeille et peuvent être restaurés.

### Accéder à la Corbeille

1. Menu "Corbeille"
2. Visualiser les courriers supprimés

### Restaurer un Courrier

1. Ouvrir la corbeille
2. Sélectionner le courrier
3. Cliquer sur "Restaurer"

### Suppression Définitive

Seuls les super administrateurs peuvent supprimer définitivement les courriers de la corbeille.

---

## Export de Données

### Export PDF

1. Ouvrir le détail d'un courrier
2. Cliquer sur "Exporter en PDF"
3. Le document est téléchargé

### Export Liste

1. Aller dans la recherche
2. Filtrer les courriers souhaités
3. Cliquer sur "Exporter"
4. Choisir le format (PDF ou Excel)

## Langues

2. Choisir la langue souhaitée

La préférence est sauvegardée dans votre profil.

## Bonnes Pratiques

1. **Sécurité** : Ne jamais partager vos identifiants de connexion
2. **Organisation** : Mettre à jour les statuts des courriers régulièrement
3. **Documentation** : Remplir tous les champs obligatoires avec précision
4. **Sauvegarde** : Les administrateurs doivent effectuer des sauvegardes régulières
5. **Déconnexion** : Se déconnecter après chaque session sur un poste partagé
